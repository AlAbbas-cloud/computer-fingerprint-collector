# Application Requirements — Computer Fingerprint Collector

## Purpose
Collect and store key system and network information from all Midtown IT computers into a CSV file.

## Data Collected
- Computer name  
- IP address  
- MAC address  
- Processor model  
- Operating system  
- System time  
- Internet speed  
- Active ports  
- Timestamp  
- Unique computer ID (if available)  

## Constraints
- Must run on Windows and Linux  
- Python 3.x only  
- Standard library only (no third‑party packages)  
- Defensive coding style
